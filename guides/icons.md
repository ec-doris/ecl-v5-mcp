# Icons

ECL EC v5.0.1 icons are rendered by WebTools classes on a span. Use the icon name in both the WebTools and ECL icon classes.

```html
<span
  class="wt-icon--audio ecl-icon ecl-icon--m ecl-icon--audio"
  aria-hidden="true"
></span>
```

For flags, use the flag code as the icon name. For network icons, use the network name as the icon name and add a WebTools style class when needed, such as `wt-icon--monochrome`.
