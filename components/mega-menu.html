⚠️ Warning: Mega Menu Component

This is an extremely complicated component. It needs to be positioned at the top of the page under a site header. There are overlays and expandables that expect to be in this position.

Users should only use a mega menu if they HAVE to. Try to use just the normal menu component almost always.
