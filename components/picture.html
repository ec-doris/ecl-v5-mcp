Basic Picture with Responsive Sources

The picture element provides responsive images using multiple source elements with media queries. The browser selects the appropriate source based on viewport size.

Picture with Zoom Effect

Add the ecl-picture--zoom modifier class to enable a zoom animation on hover or focus.

Picture with Lazy Loading

The loading="lazy" attribute defers loading of images until they are near the viewport, improving page performance.

Simple Picture without Sources

A picture element can contain just an img element without source elements for simple use cases.

Picture with Image Anchor

The image anchor controls the initial position when the image is cropped. Use the --ecl-image-anchor CSS custom property to set values like 'top left', '30% 50%', or '100px 20px'.

Picture with Extra Classes and Attributes

Custom classes and data attributes can be added to both the picture and img elements for additional styling or functionality.

Picture with Empty Alt (Decorative Image)

For decorative images that don't convey meaningful content, use an empty alt attribute to hide the image from screen readers.
